# Fiscal vs Sonegador

Um simulador de auditoria fiscal onde você deve cruzar dados financeiros para identificar sonegação de impostos.

## Tecnologias
- React
- Vite
- TypeScript

## Como Jogar
1. Instale as dependências: `npm install`
2. Inicie o servidor: `npm run dev`
3. Analise os dados das empresas e decida entre **Regular** ou **Fiscalizar**.

Para ver as regras detalhadas, consulte o arquivo [regras_do_jogo.md](./regras_do_jogo.md).
